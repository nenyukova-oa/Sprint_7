# Sprint_7 тестирование API учебного сервиса Яндекс Самокат
Файлы в папке с тестами:

1. test_create_courier: тесты для регистрации курьера

2. test_login_courier: тесты авторизации курьера

3. test_order: тесты создания заказа и вывода списка заказов

Файлы в проекте:
1. create_courier: методы для тестов регистрации курьера
2. login_courier: методы для тестов авторизации курьера
3. order: методы для тестов создания заказа и вывода списка заказов
4. generator: данные для генерации данных пользователя
3. data: ожидаемые ответы и данные для создания заказа
4. conftest: фикстуры
5. url: ссылки для тестирования сайта
